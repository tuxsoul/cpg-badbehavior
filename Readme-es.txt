= Requerimientos =

  * Coppermine Gallery 1.4 +
  * PHP4 +

= Características =

  * Capacidad de usar Bad-Behavior en Coppermine Gallery como Plugin
  * Elimina visitas indeseadas (Bots, Spammers, etc., etc.)

= Detalles =

  * Primero descarga y descomprime este Plugin en cualquier directorio
  * Utilizando un cliente FTP, sube la carpeta "cpg-badbehavior" en el 
    directorio nombrado Plugins, de tu galería de fotos hecha con Coppermine 
    Gallery
  * Utilizando tu navegador favorito, ve a tú galería de fotos, entra como
    Administrador
  * Verificar que en la configuración de tú galería este activada la opción
    "Habilitar Plugins"
  * Ve hacia el administrador de Plugins: "pluginmgr.php"
 
    Ej.
        http://www.example.com/pluginmgr.php
        http://www.example.com/cpg/pluginmgr.php

  * En la pantalla ya debe de aparecer este Plugin "cpg-badbehavior"
  * Instala el Plugin, dando clic en la letra "i", que se encuentra de lado 
    derecho

Por último ya solo queda esperar para bloquear a los spammer's que lleguen a tú
sitio web.
